# Backend E-commerce

Production-ready backend for an e-commerce platform with modular architecture, caching, and observability.

## Tech stack

- Node.js (Express)
- MongoDB
- Redis
- MinIO (S3-compatible storage)
- Nginx

## Core features

- Auth with access/refresh tokens
- Role-based access control
- Products, categories, cart, and reviews
- File uploads (multipart)
- Prometheus metrics and Sentry error tracking

## Quick start

```bash
npm install
npm run dev
```

API base URL: `http://localhost:3000/api`
