# 🚀 Feature in ScaleX-commerce

This document provides a detailed breakdown of all system features.

---

## 👤 User Features

### 🔐 Authentication & Authorization
- Secure user authentication using JWT
- Role-based access control (Buyer, Seller, Admin)
- Protected API routes using middleware

---

### 🛍️ Product Browsing
- View detailed product pages including:
  - Price, features, and description
  - Product images
  - Ratings and reviews
- Category-based browsing (10+ categories supported)

---

### 🔍 Search & Discovery
- Real-time product search using database queries
- Keyword-based search (e.g., "laptop", "phone")
- Basic recommendation system
- *(Future: AI-based recommendation engine)*

---

### 🎯 Filtering & Sorting
- Price filtering (low → high)
- Category & subcategory filters
- Easily extendable filtering system

---

### 🛒 Cart Management
- Add/remove products from cart
- Update product quantity
- View all selected items before checkout

---

### ❤️ Wishlist
- Save products for future purchase
- Move items between wishlist and cart
- Remove items from wishlist

---

### 📍 Address Management
- Add multiple delivery addresses
- Edit/delete saved addresses
- Select address during checkout
- *(Future: auto-fill via location services)*

---

### 💳 Checkout Flow
- Multi-product checkout system
- Address selection before order confirmation
- *(Future: Payment gateway integration)*

---

### ⭐ Reviews & Ratings
- Users can:
  - Rate products (0–5)
  - Write reviews/comments
  - Upload images with reviews

---

## 🧑‍💼 Seller Features

### 📦 Product Management
- Add new products with:
  - Title, description, features
  - Images
  - Category & subcategory
- Control product visibility (public/private)

---

### 📊 Inventory Management
- Track product stock levels
- Update availability dynamically

---

### 📑 Order Management
- Accept or reject incoming orders
- Monitor order status

---

### 📈 Seller Dashboard
- Overview of:
  - Total products listed
  - Sales summary
  - Inventory status
- *(Future: Advanced analytics & insights)*

---

## 🛠️ Admin Features (Planned)

- Manage all products across platform
- Remove fraudulent or unauthorized sellers
- Handle user complaints and support requests
- Monitor and control order lifecycle
